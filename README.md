## Getting Started

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Gabageba/beauty.git
   ```
2. Install packages
    ```sh
   yarn install
   ```
   ```sh
   npm install
   ```
3. Run project 
    ```sh
   yarn start
   ```
   or
   <br/>
   <br/>
   Build project
      ```sh
   yarn build
   ```
